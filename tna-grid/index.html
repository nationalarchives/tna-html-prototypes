The National Archives frontend toolkit

A collection of CSS and HTML elements for using as part of your application's frontend. A way of creating flexible and unique layouts whilst also maintaining consistency across TNA online.

Designers and developers can use the toolkit to:

  • create applications with a consistent look and feel
  • prototype, test and get feedback quickly and easily

Download toolkit

For more information please see our Design guide.

Usage

<!-- TNA-base core CSS -->
<link rel="stylesheet" href="https://cdn.nationalarchives.gov.uk/tna-grid/1.0/css/tna-grid.css">

<!-- TNA-base core minified CSS -->
<link rel="stylesheet" href="https://cdn.nationalarchives.gov.uk/tna-grid/1.0/css/tna-grid.min.css">
